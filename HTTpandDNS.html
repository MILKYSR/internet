This page contains information on Hypertext Transfer Protocol (HTTP) and the Domain Name System (DNS)




HTTP (Hypertext Transfer Protocol)

HTTP, or HyperText Transfer Protocol, is a system of guidelines that allows web browsers and web servers to communicate. When you type a website address, your browser uses HTTP to ask the server for the data that holds the website. The server responds by sending back the website's content, including text, images, and videos, which your browser shows on your screen. HTTP is When you go to a website, your browser sends an HTTP request to the server, and the server responds with the data you asked for.

DNS (Domain Name System)

DNS, or Domain Name System, acts like a phonebook for the internet. When you enter a website address, such as www.example.com, your computer can't find it right away. DNS translates that name into a special number known as an IP address, which computers use to find each other online. This way, DNS allows us to access websites using simple names instead of difficult numbers.
