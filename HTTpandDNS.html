This page contains information on Hypertext Transfer Protocol (HTTP) and the Domain Name System (DNS)




HTTP (Hypertext Transfer Protocol)

HTTP is the protocol for moving data on the internet. It works as a request-response system between your browser and a server. When you go to a website, your browser sends an HTTP request to the server, and the server responds with the data you asked for.

DNS (Domain Name System)

DNS is like the internet's phonebook. It changes domain names (such as www.example.com) into IP addresses, which computers use to recognize one another on the network.
